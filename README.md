# ⏳ Circular Loading Animation UI

Uma interface moderna de animação de carregamento circular desenvolvida com foco em simplicidade, desempenho e experiência do usuário. O projeto apresenta um spinner minimalista com rotação contínua, ideal para indicar processos em execução, carregamento de páginas e requisições assíncronas.

---

## 📸 Preview do Projeto

Interface contendo:

- Spinner circular animado
- Efeito de rotação contínua
- Design minimalista
- Animação suave e fluida
- Layout centralizado
- Estrutura leve e moderna

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

## 🚀 Objetivo do Projeto

Praticar habilidades de desenvolvimento front-end através da criação de um componente de carregamento moderno, explorando conceitos de animações CSS, transformações, keyframes e feedback visual para o usuário.

## 📚 Funcionalidades
